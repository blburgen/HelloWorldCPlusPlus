# Overview

This project implements both parts of day 1, and day 2 of the Advent of code from 2024. All the implementations were done using C++.

This project was selected to learn the basics of C++ using a set of problems to solve.

# Development Environment

- C++
- Visual Studio Code - including C++ developer extension
